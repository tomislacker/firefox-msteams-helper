# Teams Hotkey Helper - Store Description

## Short Description

Quick keyboard shortcuts for Microsoft Teams reactions and hand raise functionality.

## Full Description

Tired of clicking through menus in Microsoft Teams? **Teams Hotkey Helper** brings fast keyboard shortcuts directly to your Teams meetings and calls.

### Features

- **Lightning-fast reactions**: Trigger reactions with simple keyboard shortcuts
- **Hand raise shortcut**: Raise or lower your hand without touching the mouse
- **Non-intrusive**: Runs silently in the background on Microsoft Teams only
- **Privacy-first**: No data collection, no tracking, completely local

### Keyboard Shortcuts

Use **Shift+Alt** followed by:

- **Shift+Alt+!** (Shift+1) - Thumbs up reaction
- **Shift+Alt+@** (Shift+2) - Heart/love reaction
- **Shift+Alt+#** (Shift+3) - Applause reaction
- **Shift+Alt+$** (Shift+4) - Laugh reaction
- **Shift+Alt+%** (Shift+5) - Surprised reaction
- **Shift+Alt+~** (Shift+``) - Toggle hand raise

### How It Works

Simply press the keyboard shortcut while in a Teams meeting. The extension will:

1. Automatically open the reactions menu if needed
1. Click the selected reaction
1. Close the menu when done

### Requirements

- Works on teams.microsoft.com (web version)
- No additional permissions or configuration needed
- No personal data collection

### Privacy

This extension operates entirely within your browser. It does not collect, store, or transmit any data. See our [privacy policy](PRIVACY_POLICY.md) for details.
